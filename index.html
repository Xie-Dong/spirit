Spirit Biobanking

a study / biosample management software for drug discovery research

Start Demo (webstart)

Origin of the Spirit Biobanking project

Spirit is a biobanking application, developed by and for Actelion Pharmaceuticals, Allschwil Switzerland. The primary objective was to help scientists/technicians doing their studies. It covers all aspects from study design to biosample management, location handling and result analysis.

Demo

Click Spirit Biobank to start the program as a Webstart application. Be sure to have Java1.7+ installed and to accept the certificate.

Repository

Spirit Biobanking is released under the GPL.v3 license. As such, you can download the source and binaries of the latest release under GitHub
